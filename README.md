# GameAPI
Proyecto: API de Videojuegos con Patrones de Diseño en .NET

Descripción

Este proyecto consiste en el desarrollo de una API en .NET para gestionar la creación y recuperación de superhéroes dentro de un videojuego. La aplicación utiliza los patrones de diseño Factory y Singleton.

Tecnologías Utilizadas

    .NET 8
    C#
    Patrones de diseño: Factory y Singleton
    Swagger para la documentación de la API
